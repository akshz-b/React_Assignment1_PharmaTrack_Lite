# PharmaTrack Lite

A simple pharmaceutical inventory management system built with React.

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the app:
```bash
npm start
```

3. Open http://localhost:3000

## Features

- View all medicines in a table
- Search medicines by name or category
- Sort by expiry date and stock
- View medicine details
- Add new medicine with validation
- Highlight expiring medicines
- Show low stock badge

## Technology

- React 18
- React Router 6
